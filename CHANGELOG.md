# Changelog

All notable changes to this project will be documented here.

## [1.0.0] — 2025

### Added
- Initial release
- 20 questions across four Ikigai dimensions (Love, Mastery, Livelihood, Mission)
- Three question formats: multiple choice, 5-point Likert, emoji/icon selection
- Interactive SVG Venn diagram with clickable areas and personalised tooltips
- Score-adaptive synthesis text (12 distinct narrative variants)
- Four individual reflection cards per dimension
- Progress auto-saved via `localStorage` with session restore prompt
- Social sharing (Twitter/X, LinkedIn, Facebook) with customisable message
- Fully responsive layout (mobile, tablet, desktop)
- Wabi-sabi Japanese aesthetic (Cormorant Garamond + Zen Kaku Gothic New)
- Animated score bars and smooth question transitions
- Zero external dependencies (single HTML file)
